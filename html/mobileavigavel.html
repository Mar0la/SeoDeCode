Mobile Amigável

O que e Mobile Friendly ou Mobile Amigável?

Mobile Friendly quer dizer que todas as páginas carregam bem em dispositivos móveis. Um usuário pode acessar e navegar de uma para outra quantas vezes quiser, sem problemas de carregamento ou experiência.

O aumento de acesso ao mobile chega a ser exponencial. Para dar uma olhada nas estatística click aqui!

Quais os requisitos para um site Mobile Friendly?

  • Navegabilidade

  • Carregamento rápido

  • Responsividade

  • Controle por toque

Quando falamos de navegabilidade, referimo-nos a um site com fácil navegação.

Sobre carregamento rápido, o melhor é que seu site atinja o objetivo de carregar em, no máximo, 3 segundos, ja que pesquisas apontam que utilizadores abandonam o site apos esse período.

Dispositivos móveis têm uma lógica diferente do desktop. Inclusive no que se refere ao tamanho das telas. Por isso, é preciso que seus designers pensem e planejem o layout do site sobre isso. Uma boa pratica é utilizar metodologia Mobile Frist assim o deixando responsivo.

Como todos os smartphones funcionam controle por toque ou touch screen apenas deve se dizer que o Mobile Friendly deve ser sempre otimizado.
